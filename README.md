# 🏠 Dotfiles

My personal configuration files for Arch Linux with an optimized development environment.

## 📋 Contents

This repository contains my personal configuration files for:

- **Terminal & Shell**: Bash/zsh configurations and command-line utilities
- **Text Editors**: Configurations for editors and IDEs
- **File Manager**: [Yazi](https://github.com/sxyazi/yazi) with custom themes
- **System**: Arch Linux specific configurations

## 🚀 Installation

### Manual Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/dotfiles.git ~/.dotfiles
```

2. Create symbolic links for the configurations you need:

```bash
# Example for Yazi
ln -sf ~/.dotfiles/.config/yazi ~/.config/yazi
```
